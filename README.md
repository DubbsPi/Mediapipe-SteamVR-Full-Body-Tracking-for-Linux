**A SteamVR tracking solution that uses mediapipe to run**\
Works with ALVR and phone VR

-- Requirements:\
Installed programs: SteamVR\
System requirement: python3.12\
Python requirements: opencv-python, mediapipe, Pillow, pygame, argparse, scipy, numpy-quaternion

-- To install:\
Run install.sh\
It will install all requirements (aside from SteamVR) and drivers

-- To run:\
Run open SteamVR and then run the run.sh\
Some config values such as the camera require a restart to take effect

-- To uninstall:\
Run uninstall.sh\
It will remove the driver and python environment
